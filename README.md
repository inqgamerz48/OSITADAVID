# OSITA DAVID - Official Platform

> The premium digital experience for Osita David.

This platform serves as the official digital hub for Osita David, integrating his music, merchandise, exclusive drops, and tour access into a seamless, high-end web experience.

## Overview

The Osita David web platform is built with a focus on luxury aesthetics, high performance, and deep fan engagement. It features cinematic micro-animations, glassmorphism UI elements, and a dark-mode-first design language.

### Key Features
- **Cinematic Video Display**: Embedded music video features showcasing his latest releases.
- **Exclusive Access**: A dedicated funnel for fans to unlock unreleased tracks and behind-the-scenes content.
- **Integrated Shop**: A premium product showcase for physical and digital merchandise.
- **Music Integrations**: Direct connections to his Spotify, Apple Music, YouTube, SoundCloud, and Instagram.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS, CSS Variables
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

To run the project locally:

```bash
# 1. Install dependencies
npm install

# 2. Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
